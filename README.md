# バトルログ Champions 使い方

ポケモン対戦の勝敗、使用パーティ、メモを、PCとスマホで共通して記録・振り返りできる個人用Webアプリです。

## 使い始める

1. 公開URLをPCまたはスマホのブラウザで開きます。
2. 登録済みのメールアドレスとパスワードを入力して、ログインします。
3. 最初に使う端末では、ログイン後に少し待つと保存済みの記録が表示されます。

スマホでは公開URLをホーム画面に追加しておくと、アプリのようにすぐ開けます。

## 試合を記録する

1. 画面上部の「記録」を開きます。
2. 勝敗、日時、シーズン、使用したパーティ、メモなどを入力します。
3. 保存ボタンを押します。
4. 画面に「クラウドに保存中」または保存完了の表示が出ていることを確認します。

保存後は、同じアカウントでログインしているPC・スマホの両方で同じ記録を確認できます。

## シーズンとパーティを管理する

- 「シーズン」で、対戦期間ごとにシーズン名を追加・編集します。
- 「パーティ」で、よく使う6匹の組み合わせをプリセットとして登録します。
- 試合を記録するときに、作成したシーズンやパーティを選べます。

## 振り返る

「振り返り」を開くと、登録した対戦記録をもとに勝敗や傾向を確認できます。記録を増やすほど、振り返りの内容も正確になります。

## 使用上の注意

- **同じメールアドレス・パスワードを、他人に教えないでください。** 知っている人はあなたとしてログインできます。
- 記録の保存先はクラウドです。保存表示が終わる前にページを閉じたり、通信を切ったりしないでください。
- 複数の端末で同時に同じ記録を編集すると、後から保存した内容で上書きされる可能性があります。一方の端末で保存したら、もう一方はページを再読み込みしてから編集してください。
- ログアウトした端末では、次回利用時にメールアドレスとパスワードが必要です。
- Supabaseの無料プランでは、長期間使わないプロジェクトが一時停止される場合があります。その場合でも、再開後にデータベースは復帰します。大切な記録は、ときどき動作を確認してください。
- このリポジトリには対戦記録を登録しません。`data/` と `.runtime/` はGitHubへの登録対象から除外されています。

# おまけ：このアプリの仕組み

## 全体像

このアプリは、役割の違う3つの場所を組み合わせて動いています。

| 場所 | 役割 | たとえると |
| --- | --- | --- |
| GitHub Pages | 画面やプログラムを公開する | アプリを置く配布棚 |
| Supabase Authentication | ログインを確認する | 本人確認をする受付 |
| Supabase Database | 対戦記録を保存する | 鍵付きの記録庫 |

PCでもスマホでも、公開URLを開くとGitHub Pagesから画面のプログラムが届きます。その後、Supabaseにログインし、記録を読み書きします。

## アカウントとパスワード

アプリ用のメールアドレスとパスワードは、Supabase Authenticationで管理されています。これはGitHubのアカウントやSupabaseの管理画面へ入るためのパスワードとは別物です。

パスワードそのものは、アプリのコードやGitHubには書かれていません。Supabase側で、元の文字列に戻せない形に変換して保管されています。そのため、パスワードを忘れた場合は「表示して確認」ではなく、再設定が必要です。

## 対戦記録の保存場所

公開版で新しく保存した対戦記録は、Supabase Databaseの `battle_log_data` テーブルに保存されます。ここには、対戦記録、シーズン、パーティプリセットがまとめて入っています。

PCで保存したデータをスマホでも見られるのは、PCからスマホへ直接送っているのではなく、どちらも同じSupabaseの記録庫を見に行くからです。

ローカルの `data/battle-log.json` は、開発中の確認や昔のローカル記録の控えです。GitHub Pagesの公開版はこれを使用しません。また、`.gitignore` の設定によりGitHubへ公開されません。

## 他人に見られない理由

データベースにはRLS（行レベルセキュリティ）というルールを設定しています。これは「ログインした人は、自分の利用者IDが付いた記録だけを読んだり変更したりできる」という門番です。

つまり、公開URLを知っているだけでは記録を読めません。ログインしても、別のアカウントならあなたの記録にはアクセスできません。

## ログアウトの意味

ログアウトは、その端末に保存されているログイン状態を消す操作です。PCでログアウトしても、スマホのログイン状態は消えません。共用PCや他人の端末では、使い終わったらログアウトしてください。
